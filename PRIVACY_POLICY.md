# Privacy Policy

Коротко: **moysklad-mcp-ru — локальный инструмент. Он не собирает и не передаёт
ваши данные.**

## Что обрабатывается и где

- **API-токен МойСклад** хранится локально в `~/.moysklad-mcp/cabinets.json`
  (chmod 600). Не передаётся никуда, кроме самого API МойСклад (в заголовке
  авторизации запросов, которые инициирует ваш агент).
- **Данные учёта** (остатки, заказы, документы и т.п.) приходят напрямую из API
  МойСклад в вашего агента и обрабатываются на вашей машине. Сервер их не
  сохраняет и не пересылает.

## Чего сервер НЕ делает

- Не отправляет телеметрию и аналитику.
- Не «звонит домой», не имеет внешних сервисов автора.
- Не передаёт токен или данные третьим лицам.
- Не пишет секреты в репозиторий, в конфиг клиента или в чат.

## Третьи стороны

- **МойСклад** — ваши запросы идут в его API по вашему токену; применяется
  политика МойСклад.
- **Ваш ИИ-клиент** (Claude, Cursor, Codex и т.д.) — текст диалога обрабатывается по
  политике соответствующего провайдера. Не вставляйте токен в чат: тулы спроектированы
  так, чтобы секрет туда не попадал.

## Контроль

Токен и кабинеты — ваши локальные файлы; удаляются `ms_remove_cabinet` или
удалением `~/.moysklad-mcp/cabinets.json`. Копия приложения — в `~/.moysklad-mcp/`,
удаляется вручную.
